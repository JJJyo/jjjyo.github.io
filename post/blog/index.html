博客建立命令

Aug 7, 2019
1 min read

##关于hugo cd ~/myblog/

hugo server -t m10c –buildDrafts 本地启动服务

hugo new post/xxx.md 在/content/post下建了xxx.md

hugo –theme=m10c –baseUrl=“https/jjjyo.github.io” –buildDrafts 更新

##关于git 正确步骤： git init //初始化仓库

进入public git add .(文件name) //添加文件到本地仓库

git commit -m “first commit” //添加文件描述信息

git remote add origin + 远程仓库地址 //链接远程仓库，创建主分支 https://github/JJJyo/jjjyo.github.io.git

git pull origin master // 把本地仓库的变化连接到远程仓库主分支

git push -f origin master //把本地仓库的文件推送到远程仓库

更新博客

cd ~/myblog/public/

git add .(文件name) //添加文件到本地仓库

git commit -m “first commit” //添加文件描述信息

git push -u origin master //把本地仓库的文件推送到远程仓库
